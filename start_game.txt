Добро пожаловать в систему управления базой данных игр

Перед запуском у тебя должен быть заранее скачен python.
Скачать все нужные библиотеки --> pip install -r requirements.txt
Что бы запустить игру запусти cmd и перейди в директорию с файлом. Перейдя в директорию пропиши --> python main.py
Приятного использования

Немного о системе:

Добавление игры (add_game): Пользователь вводит информацию о игре: название, издатель, год издания. Данные сохраняются в базу.

Поиск игры (search_game): По заданным параметрам (название, издатель, год издания) производится поиск соответствующих игр в базе.

Удаление игры (del_game): Пользователь может удалить игру из базы по заданным параметрам.

Редактирование данных о игре (edit_game): Пользователь может изменить данные о игре в базе.

Вывод списка всех игр (list_games): Выводит список всех игр в базе.
